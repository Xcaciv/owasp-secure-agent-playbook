# S7 Roles and Responsibilities in SSEM Adoption

## S7.1 Empowering Senior Software Engineers

Experienced, top-tier software engineers are critical to the success of any AppSec program. AppSec professionals should work collaboratively with these engineers in design activities.

### Senior Engineers Should Be Empowered To

| Responsibility | Description |
| :--- | :--- |
| Ask "What can go wrong?" | Consider security issues at every stage of development |
| Drive security requirements | Create Security Requirements, Acceptance Criteria, and Threat Scenarios |
| Lead merge reviews | Emphasize SSEM attributes and software engineering principles |
| Champion dependency maintenance | Schedule and lead regular dependency updates |
| Mentor peers | Guide less experienced developers in securable coding practices |
| Liaise with AppSec | Ensure smooth security alignment with software engineering practices |
